Mijn Projecten

GPS Tracker

Voor het vak Systems Engineering & Analysis (SE&A) heb ik samen met 4 medestudenten een gps-tracker gemaakt. Onze klant kwam naar ons met de vraag of we een tracking systeem konden maken zodat ze kan kijken waar haar zoontje/dochter was als ze bijvoorbeeld naar een speeltuin gaan.

Als resultaat van deze opdracht hebben we een armbandje gemaakt met een gps tracker in die het zoontje/dochter makkelijk kan aandoen. de locatie wordt gedeelt op onze zelfgemaakte website met een accurate lokatie die om de 3 seconde reset. We hebben ook een paar knoppen toegevoegd die onze klant kan indrukken op de pagina om haar zoontje/dochter op te roepen.

Hosting Platform

Voor het vak Project Hosting hebben ik en 4 medestudenten een hosting platform gemaakt. We kregen de opdracht om een een platform te maken waar de studenten van het derde jaar Application Development hun project kunnen op hosten. Onze servers staan al volledig klaar in het serverrek en we moeten ons nu alleen nog op de software focussen. Hier gaan we verschillende tools voor gebruiken zoals: LDAP, API, SFTP & Auth0. Momenteel ben ik aan de webpagina aan het werken zodat onze klanten hun bestanden zonder problemen kunnen uplaoden.

Muizenval

Vorig jaar heb ik voor het vak Internet of Things (IoT) een muizenval ontworpen. Het doel is simpel, ik moet een muis kunnen vangen. Dit heb ik gedaan aan de hand van verschillende componenten: OrangePi, stappenmotor, knop, LED, afstandssensor & LCD. Wanneer er iets in mijn muizenval kwam pakte de afstandssensor dit op en ging de stappenmotor draaien waardoor het luik dicht valt.

Natuurlijk wil ik alleen een muis vangen dus heb ik er Artificial intelligence (AI) in gestopt. Aan de hand van een camera en 100 afbeeldingen van muizen heb ik de AI laten trainen wanneer de camera een muis opneemt zal de AI dit herkennen en de het luik dichtvallen.

Webstack on Kubernetes

Vorig semester heb ik voor het vak Linux Webservices een Webstack in Kubernetes gemaakt. Ik heb hier aan de hand van docker en kubernetes 2 containers aangemaakt. Een Apache2-container die een webpagina bedient en een MySQL-container die een database host met één enkel record waarin mijn volledige naam staat. Deze containers heb ik verbonden aan de hand van een API. In dit project haal ik mijn naam op de Apache2-webserver uit de MySQL database op de andere container. Als ik mijn naam in de database zou veranderen, update de webpage dit automatisch. Ook als ik mijn containers verwijder en opnieuw op zet, gaat de data niet verloren.
